# Results Directory

This directory contains per-seed AUROC outputs from all experiments.
Files are not tracked by git (see `.gitignore`) and must be regenerated by running the experiment scripts.

## File Formats

### `main_W{W}_pca{K}.csv`
Generated by `experiments/run_main.py`.

| Column | Description |
|--------|-------------|
| `seed` | Random seed index (0–19) |
| `phase_auc` | AUROC for phase cross-spectral + PCA(K) + LR |
| `voss_auc` | AUROC for Voss power spectrum + PCA(K) + LR |

### `ablation_multiscale.csv`
Generated by `experiments/run_ablation.py` (Exp B).

| Column | Description |
|--------|-------------|
| `seed` | Random seed index |
| `multiscale_auc` | AUROC for W=90+W=420 concatenated PCA-32 + LR |

### `backtranslation_W{W}.csv`
Generated by `experiments/run_backtranslation.py`.

| Column | Description |
|--------|-------------|
| `seed` | Random seed index |
| `real_cds` | AUROC on real CDS sequences only |
| `all_seqs` | AUROC including back-translated sequences |
| `delta` | `real_cds - all_seqs` (positive = back-translation hurts) |

## Reproducing Results

```bash
# Table 1, W=90
python experiments/run_main.py --input data/sequences.csv --w 90 --pca 64

# Table 1, W=420
python experiments/run_main.py --input data/sequences.csv --w 420 --pca 128

# Table 2 ablations
python experiments/run_ablation.py --input data/sequences.csv

# Back-translation sensitivity
python experiments/run_backtranslation.py \
    --input data/sequences.csv \
    --backtranslated data/backtranslated.txt \
    --w 90 --pca 64
```
