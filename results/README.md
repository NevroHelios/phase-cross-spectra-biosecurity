# Results

Pre-computed outputs from all experiments. Regenerate by running the scripts from the repo root.

## Files

### `stats_tables.csv`
Generated by `experiments/exp_stats_analysis.py`. Primary output — **Table 2** of the paper.

| Column | Description |
|--------|-------------|
| `config` | Feature configuration name |
| `n_seeds` | Number of seeds (20) |
| `testA_auroc` / `testA_auroc_std` | Mean ± std AUROC on gene-family holdout |
| `testB_auroc` / `testB_auroc_std` | Mean ± std AUROC on 5 withheld genera |
| `testA_tpr1fpr` / `testA_tpr1fpr_std` | TPR at 1% FPR on TestA |
| `testB_tpr1fpr` / `testB_tpr1fpr_std` | TPR at 1% FPR on TestB |
| `gap_mean` / `gap_std` | TestA − TestB per seed, mean ± std |
| `gap_ci_lo` / `gap_ci_hi` | 95% CI on gap (t-distribution) |
| `wilcoxon_W` / `wilcoxon_p` | Two-sided Wilcoxon signed-rank test on 20 gaps |

### `stats_raw_arrays.npz`
Generated by `experiments/exp_stats_analysis.py`. Per-seed raw arrays for all 6 configs. Keys follow the pattern `{config_key}_testA` / `{config_key}_testB`.

### `testa_aucs_codon_unigram_lr.npy` / `testb_aucs_codon_unigram_lr.npy`
20-element arrays of per-seed AUROC for the primary configuration (codon unigram 64d, logistic regression).

### `testb_aucs_xgb_best.npy`
20-element array of per-seed TestB AUROC for XGBoost d=8.

### `full_eval_summary.csv`
Generated by `experiments/exp_full_eval.py`. Phase+Voss W=110 PCA-128 on same-strain gene-family holdout — **Table 1 Stage 3** reference (0.682).

| Column | Description |
|--------|-------------|
| `test_set` | `A_same_strain_holdout` or `B_novel_genera` |
| `auroc_mean` | Mean AUROC over 20 seeds |
| `auroc_std` | Std AUROC over 20 seeds |
| `description` | Human-readable label |

## Reproducing

```bash
# Table 2 — all feature configs
python experiments/exp_stats_analysis.py

# Table 1 Stage 3 — Phase+Voss same-strain baseline
python experiments/exp_full_eval.py
```
